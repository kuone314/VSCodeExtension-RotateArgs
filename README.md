# Rotate Args


![Sample](Doc/Demonstration.gif)
* Rotate strings with a delimiter(comma,space,=) for the selection.  
  選択範囲に対し区切り文字(comma,space,=)で文字列をローテーションする。

<!-- ## Usage -->

## Commands
* `rotate-args.byRegExp`
* `rotate-args.byComma`
* `rotate-args.byEqual`
* `rotate-args.bySpace`
